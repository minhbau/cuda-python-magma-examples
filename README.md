# cuda-python-magma-examples

This repository contains some examples for running eigenvalue solver GEEV with GPU using [scikit-cuda](https://github.com/lebedov/scikit-cuda).

## Files:
 
```
gpu-linalg.py 
magma_all_geev.py 
magma_zgeev.py
```

[gpu-linalg.py](https://github.com/wingkitlee0/cuda-python-magma-examples/blob/master/gpu-linalg.py) This is a simple wrapper for **eig** routiner in scipy.
